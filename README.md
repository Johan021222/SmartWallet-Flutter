# 💰 SmartWallet - Gestión de Finanzas Personales

**SmartWallet** es una solución multiplataforma diseñada para ayudar a los usuarios a tomar el control de su dinero. Permite registrar ingresos y egresos de forma sencilla, categorizar gastos y visualizar el estado financiero actual mediante una interfaz moderna y limpia.

Este proyecto forma parte del portafolio de desarrollo de software para el semestre final de carrera.

---

## 🎯 Objetivo del Proyecto
El propósito es aplicar conceptos de **persistencia de datos locales** y **diseño de interfaces de usuario (UI/UX)** para crear una herramienta útil que funcione sin necesidad de internet, manteniendo la integridad y privacidad de la información financiera del usuario.

## 🛠️ Tecnologías y Herramientas
Para este proyecto estamos utilizando un stack moderno de desarrollo móvil:

* **Lenguaje:** [Dart](https://dart.dev/) (Tipado fuerte y optimizado para UI).
* **Framework:** [Flutter](https://flutter.dev/) (Para renderizado nativo en Web, Android e iOS).
* **Sistema de Diseño:** **Material Design 3** (La versión más reciente del lenguaje visual de Google).
* **Control de Versiones:** **Git** y **GitHub** para la colaboración y respaldo.
* **Persistencia:** **SQLite** (Base de datos local segura y confiable para almacenamiento de transacciones).
* **Librerías:** `sqflite` para manejo de SQLite y `path_provider` para acceso a rutas del sistema.
